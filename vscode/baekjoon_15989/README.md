### 입력
```
3
4
7
10
```

### 출력
```
4
8
14
```

### 점화식 설계
dp[i][j] : 1~j까지의 수 조합으로 i 생성 가능한 경우의 수(j 반드시 포함)

ex. 
```
dp[1][1] = 1 | 1
dp[1][2] = 0 |
dp[1][3] = 0 |

dp[2][1] = 1 | 1+1
dp[2][2] = 1 | 2
dp[2][3] = 0 |

dp[3][1] = 1 | 1+1+1
dp[3][2] = 1 | 2+1
dp[3][3] = 1 | 3

dp[4][1] = 1 | 1+1+1+1
dp[4][2] = 2 | 2+1+1, 2+2
dp[4][3] = 1 | 3+1

dp[5][1] = 1 | 1+1+1+1+1
dp[5][2] = 2 | 2+1+1+1, 2+2+1
dp[5][3] = 2 | 3+1+1, 3+2
```
i>3인 경우, 다음 점화식이 성립한다.
```
dp[i][1] = dp[i-1][1]
dp[i][2] = dp[i-2][1] + dp[i-2][2]
dp[i][3] = dp[i-3][1] + dp[i-3][2] + dp[i-3][3]
```

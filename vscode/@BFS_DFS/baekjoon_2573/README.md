[#2573 빙산](https://www.acmicpc.net/problem/2573)
---

- BFS
- challenge1 : 주변 영토의 특성{바다/육지} 따라 다른 조건 수행 (+ visited 관리)
- challenge2 : 바다에 의한 녹음 현상 한 번에 수행 \
    -> BFS 중간에 0이 되어 녹은 빙산의 바다 역할 방지하기 위함

---

### 입력
```
5 7
0 0 0 0 0 0 0
0 2 4 5 3 0 0
0 3 0 2 5 2 0
0 7 6 2 4 0 0
0 0 0 0 0 0 0
```

---
### 출력
```
2
```

- floyd-warshall : 모든 지점에서 다른 모든 지점까지의 최단 경로 모두 구하기 \
$D_{ab} = \min(D_{ab},\; D_{ak} + D_{kb})$- dijkstra

- floyd-warshall vs dijkstra
    | 알고리즘           | 구하는 것               |                 시간복잡도 | 장점                               | 단점                              |
    | -------------- | ------------------- | --------------------: | -------------------------------- | ------------------------------- |
    | Floyd-Warshall | 모든 정점 → 모든 정점 최단 거리 |              (O(N^3)) | 구현이 단순함, 모든 쌍 최단 거리 문제에 바로 적용 가능 | 정점 수가 많아지면 비효율적                 |
    | Dijkstra       | 한 정점 → 모든 정점 최단 거리  |         (O(E \log V)) | 희소 그래프에서 빠름, 우선순위 큐로 효율적         | 모든 쌍 최단 거리를 구하려면 `N`번 실행해야 함    |
    | Dijkstra (N)번  | 모든 정점 → 모든 정점 최단 거리 | (O(N \cdot E \log V)) | 간선이 적은 그래프에서는 Floyd보다 유리할 수 있음   | 구현이 더 복잡하고, dense graph에서는 비효율적 |
